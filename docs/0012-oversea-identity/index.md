## 前言

作为世界公民，人可以不在国外，但国外的优质服务能享受则享受，你的视野决定你的人生去向。

## 海外住宅 IP

稳定纯净的静态住宅 IP 是不被封号的关键

IP 类型：ISP (Internet Service Provider) 而非 IDC (Internet Data Center)（易被识别为机器人从而被封）

> 购买海外 VPS 自己搭梯子就是 IDC
>
> 原生 IP：IP 注册地和机房所在地一致，反之则是广播 IP (Netflix 可能会看重)

非机房 IP 包括：动态住宅 IP、静态住宅 IP、移动流量 IP 

ip 的共享人数 & 前人作为影响 ip 的纯净度

机场一般是动态共享的机房 ip，日常使用静态共享少的住宅 ip （退而求其次的动态住宅 IP 也行）就可以应对大多数场景了

> IP 检验极其严格的才需要 VPS 自建、指纹浏览器、远程桌面等方式
> 
> 搬瓦工（Bandwagon Host）：一家加拿大 VPS 提供商，早年价格低廉，现在提供极高质量的直连线路（CN2 GIA, Softabank, HK/JP GIA），太知名以至于是重点关注对象

### 网站工具

* 查询 ip 是住宅 ip 还是机房 ip：ipinfo.io 的 dashboard
* 查看是否是原生 IP：bgp.he.net
* 查看 ip 纯净度：ping0.cc
* 查看 ip 风险：scamalytics.com 
* 测量网络带宽：speedtest.net
* 查看是否有 DNS 泄露：dnsleaktest.com

### 机场购买指南

心理价位说明：小于 ¥20/月 以下 100 G 左右适用于绝大多数普通人场景

任何机场和梯子都有跑路风险，建议购买月付、备用若干、及时更换

> 主力 VPN 月付，备用机场按量付费

筛选条件：运营时间超过两年，有官方 Telegram 群组，完善工单系统，包含常用节点（香港、新加坡、日本、美国）

大部分节点延迟在 150ms 内可接受（尤其在晚高峰中用油管测试）

线路类型：

1. IEPL/IPLC 专线：物理专线不经公网，极其稳定延迟极低，成本较高
    * 路径极简（通常为 国内跳点 → 境外入口IP，中间无公网节点）。
    * 30ms 内，5 跳以内，高峰丢包率 <1%
2. CN2 GIA/BGP中转：优质公网线路，稳定性较好，性价比高
    * traceroute 路径中出现 59.43.x.x（CN2骨干网IP）
    * CN2线路的ASN为 AS4809，普通电信163网络为 AS4134，阿里云/腾讯云IP 为 AS37963
    * 100ms 内，10 跳以内，高峰丢包率 <10%
3. 普通/直连/隧道：廉价机场常见，晚高峰易爆炸
    * 200ms 内，大于 12 跳，高峰丢包率 >10%

网络协议类型

* Shadowsocks： 只传输加密后的二进制流，最早普及，延迟极低，跨平台支持好，如今容易被封
* trojan：伪装成 https，隐蔽性强，当前主流
* VLESS：极致性能，安全性极高，但配置复杂
* XUDP: 对 UDP 流量的优化传输技术
* Hysteria2 采用了一种极其激进的拥塞控制 QUIC 算法

> 虚拟网卡 / tunnel 模式会接管本机所有软件流量（有些软件默认不走代理比如终端、大型游戏、远程桌面、指纹浏览器等）
> 
> 电脑 IP 改成固定的后可以作为手机/平板的局域网代理或路由器网关

## 国际银行卡

银行卡信息：

* 发卡机构：具体银行或金融机构（比如招商银行、Wise、美国银行、汇丰银行等），关注福利比如返现、货币转换费（FTF）、App 体验等
* BIN 码：包含发卡组织（Visa）、发卡机构、卡片类型、卡片等级、国家等。有些商家可能会限定某些条件
* 发卡国家：中国发行 Visa 卡可能买不了外国特供数字商品
* 卡片类型：Credit（先花后还，最不容易被拒付）、Debit（借记卡/储蓄卡）、Prepaid（独立钱包，权重低）

> 为什么用信用卡，银行如何利用信用卡赚钱？
> 
> 「花昨天的钱」vs. 「花明天的钱」：信用卡通常有 40-50 天的免息期，聪明的用户会存入理财然后用信用卡周转，同时信用卡通过积分、返现、信用履历强化这种心理暗示。同时，部分特殊情况下用户可以申请拒付，风险由银行先扛
> 
> 银行获利方式：未按时还钱的复利（实际年化利率高于 15%）& 商户手续费更高

### 银行卡清算组织

Card Networks：银行卡必须经由这些组织才能把钱付出去

Visa & Mastercard：市场老大和老二，覆盖面极广

UnionPay（银联）中国标准，但许多国外地区暂不支持

American Express 主打高端市场（手续费高，可能会被小商户拒收）

JCB：日本旅游必备

> 清算组织不是发卡机构

### 数字银行

转为不分国界的省钱和效率而生

> 跑路风险极低，国家管控力度弱

* Wise: 汇率极低廉
    * 可用于 X 账号收款
* Revolut：除了换汇和刷卡，还能投资理财

不频繁更换 IP，不碰加密货币，不存大额资金（几百美元即可，当转账跳板）

> 建议使用护照注册

### 香港银行卡篇

人民币与外汇的出入金节点

需要肉身赴港下载相关 App 线上上传港澳通行证 + 内地身份证 + 移民局出入境记录（微信「移民局 12367」小程序下载）办理，注意免除管理费的相关措施。

* 汇丰One（HSBC）：稳定性最强、兼容性最好。
* 中银香港：划转中国银行内地无手续费

> 在没有美国住址和社会保险号SSN的情况下，办理美国银行卡比较困难

## 境外手机号

美国运营商三大巨头：Verizon, AT&T, T-Mobile

美国宽带服务和手机服务供应商不完全重合

「虚拟运营商」（mobile virtual network operator，MVNO）：没有自己的移动网络，而是租用三大运营商的实体网络，然后设计产品、打包卖给客户。

价格通常便宜 40% 以上，合约期限更短，网络服务在人多时会被限速，适合“养号”

* Ultra Mobile：国内收短信不要钱，Paygo 月租 $3-$5，适合华人养号（实体卡）
    * 淘宝 ¥200 左右，需要开启 WiFi Calling，参考[博客](https://xunihao.net/3221.html)
* Visible：背靠 Verizon，单人无限流量首选（$25-40/m）
* Google Fi：背靠 T-Mobile，适合环球旅行者，全球无感漫游体验极佳（$20 (基础) + $10/GB）
* Mint Mobile：背靠 T-Mobile，按年买便宜（$15/m）

> 虚拟运营商的稳定性较高，虚拟号比如 Google Voice 实际上是网络拨号，虽然免费但会被识别为非手机号

除了 ultra mobile paygo，性价比较高的还有

* 英国 giffgaff 卡：卡片本身免费（需跨境邮寄或找代购），开始充值 10 英镑即可（通过其他人链接还有积分），每 180 天保号一次（发条短信约 £0.3）。在国内可以漫游接收短信（免费）。支持激活后转为 eSIM（需手机支持）。
* 香港 Club SIM：首购金（约 HK$50 左右），之后每年消费一次（如买个 HK$6 的短信包）即可保号。优势：支持 eSIM，无需物理寄送，App 扫码即用。香港号码在注册海外社交媒体、绑定汇丰/中银香港账户时极其便利。注意：根据香港法例，需进行实名登记（用护照或港澳通行证，在 App 内几分钟搞定）。

实体卡可转为 esim，需要手机支持：国行 iPhone 只支持实体 sim 卡，美国 iPhone 只支持 esim，推荐加拿大版（一实体一esim）

> 对于临时使用，可以考虑一次性接码平台、Burner、esim 旅游卡，但不推荐作为长期方案

> 可以购买 ESTK 卡用于管理多张 esim（对于不支持 esim 的机型）

### 海外住宅地址

对于云玩家：在 Google Maps 找一个免税区的住宅地址（（如俄勒冈 Oregon 或特拉华 Delaware，省 10% 消费税）的非商业住宅），复制其地址和邮编（Zip Code）即可，记一个相对固定、且与你常用静态住宅 IP 一致的地址。

> 适用于不需要邮寄账单、非金融风控较低的情况

终极方案：使用在美国亲友的真实住宅地址。

### 美区 Apple ID

同理，不过需要考虑海外居住地

> Apple Gift Card 是个好东西，和 Switch2 的 Gift Card 同理

### 海外证券银行账户

> 护照是国际认可，身份证是中国认可

传统国际券商巨头：盈透（Interactive Brokers）、嘉信（Charles Schwab）

智能化互联网券商：长桥、富途等

> W-8BEN 表单：在非美国公民开设美国银行或证券账户时，务必填写此表，可以免除美国境内的利息税/资本利得税（中国居民适用 10% 的优惠预扣税率）
