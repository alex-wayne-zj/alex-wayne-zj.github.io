## 前言

数字收藏从各方面来看都更优：

* 符合我「行李越少，就越自由」的现代需求，精神世界脱离物质而存在
* 金钱消费更低、获取更方便、使用更无感、无需维护

优先使用已有设备作存储。

## 书籍

常见数字版书籍格式

* azw3 / mobi: Kindle 专用
* pdf: iPad / Mac 专用
* epub: 现代阅读器支持较好

要求：azw3, 无乱码，未删减，知名出版社，公认优秀版次，外文尽量看原版，否则精选译者

**常用软件**：ZLibrary、豆瓣、Goodreads

**存储介质**：Kindle 收藏文件夹

## 影视

要求：

* 分辨率
    * 推荐1920 * 1080，可以3840 * 2160
    * 14寸MacBook上观看：30cm内能感受到4K细节优势，50cm外1080p和4K差异并不显著
* 帧率
    * 一般都是24帧，有60fps则优先选择
* 码率
    * 越高越好，推荐50mbps左右
* 色深
    * MacBook Pro原生10bit
* HDR：HDR的色彩和亮度提升，比分辨率提升带来的视觉影响更显著，而色彩对人的影响又小于亮度
    * HDR10:相对更流行
      * 要求10bit色深，Rec.2020色域
      * 额外增加10-20%文件体积
    * Dolby Vision：更高端
* 编码标准 Codec 
    * H.265 / HEVC / AV1
* 音轨
    * Dolby TrueHD (Atmos 杜比全景声) / DTS-HD Master Audio / E-AC-3 / DTS
    * AAC 只有双声道，尽量 5.1 声道起步
* 压制源
    * REMUX（原盘提取）：没有任何二次压缩，画质完美，单部电影体积通常在 50GB - 100GB。
    * WEB-DL（流媒体直接下载）： 从 Netflix、Disney+ 等流媒体服务器上直接扒下来的原文件，无水印无广告，画质很好，体积适中（4K 电影通常 15GB - 25GB），目前性价比最高的收藏格式。
    * BDRip / WEBRip（二次压制）： 压制组为了缩小体积重新编码的版本。
* 其他要求
    * 没有博彩行业或其他广告
    * 外文电影必须有多语字幕，不能只有中文字幕
    * mkv 格式对字幕和高级音轨的兼容性比 mp4 更好

将来有条件后配置投影仪（推荐坚果三色激光）和蓝牙音箱（推荐HomePod Mini）

对极其热爱的作品，考虑收藏4KUHD光碟。

**存储介质**：笔记本和云盘空间过载后，单盘冷备

**常用软件**：片库、豆瓣、Imdb

## 音乐

* 采样率：48kHz
    * 人耳范围20Hz-20kHz
    * 奈奎斯特定理：采样率大于频率两倍
    * DAW标准：96kHz
* 位深
    * 16位深，65536个音频值
    * DAW标准：24位深
    * 有损压缩后位深信息丢失
* 声道数
    * 立体声双声道Stereo
    * DAW标准：5.1/7.1声道
* 比特率
    * 未压缩音频比特率可由前三项直接算出
    * 作为有损压缩输出音频质量参考：较高质量256kbps & 320kbps
* 编码方式
    * 无损FLAC / 有损MP3
    * Apple系 无损ALAC / 有损AAC

要求：>=44.1kHz，Stereo，至少256kbps，MP3格式（最大20MB）

> 作为参考，Apple Music 的高质量标准：AAC 编码 256kbps

**播放设备**：MacBook Pro，高保真音响（从HomePod Mini开始）

**常用软件**：网易云音乐、Apple Music、Spotify（未来）

> 通常来说，作为木耳且更看重音乐故事和情绪的听众，上述标准已然足够，但也不妨写下最高标准以供参考（这时制约音质的主要是播放设备）

**顶级数字音乐收藏标准**（几乎就是 DAW 标准）：192kHz，32位深，7.1 声道，ALAC/FLAC 无损格式

**存储介质**：笔记本和云盘空间过载后，单盘冷备（体积相对较小）

### 关于实体版

> 黑胶唱片有底噪，需要单独调整频响，单论音质是不如 CD 的。更多的是做家居摆件、满足消费主义的装逼需求。
>
> KPOP idol 专辑中通常就是 CD

光盘：使用激光技术存储数据的圆形盘片​​。包括​​CD（Compact Disc，700MB），Blu-ray Disc（蓝光光盘，单层 25G）​

> 蓝光光驱可同时读取 CD 和 Blu-ray Disc

收藏实体音乐需要的物理设备：CD、CD机、功放、线材、音响等

## 图像

用途：记录生活留下美好、Mac 壁纸、TV 壁纸、手机壁纸、iPad 壁纸、公众号/博客封面图、照片打印、收藏欣赏

要求：互联网上收藏，不带水印，1080P 以上，10 bit 位深，PNG 和 WEBP 优于 JPG

待探索网站：

* Unsplash（免费高质量摄影作品，4K/5K 多）
* Pexels（风格多样，支持按分辨率筛选）
* Wallhaven（海量高清壁纸，分类细致）
* Simple Desktops（极简风格，适合桌面清爽）

**常用软件**：小红书，Instagram
